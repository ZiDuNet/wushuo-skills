# 四层内容架构

> 所有场景生成的材料必须包含四个层次，缺一不可。
> 无论给谁看、什么格式，这四层都要有。

---

## 第一层：业务内容

课程设计、课表安排、知识库素材匹配——这是核心价值。

**通用规则：**
- 课程内容必须基于飞书知识库素材，不能编造
- 课表按 Unit 结构组织（每个 Unit = 2天 = 4个半天）
- 素材匹配原则：根据客户画像从知识库选择最匹配的文档，一个半天可组合多个文档
- 痛点和前后对比必须用客户真实场景，不用泛泛模板话

## 第二层：背景内容

> **核心目的：制造紧迫感。** 让客户意识到"不跟上 AI 就要被淘汰了"。
> 背景内容必须每次都从互联网实时检索，绝不用固定话术或过时数据。

**每次生成时，用 WebSearch 搜索以下维度的最新信息（用当前年月构造搜索词）：**

| 要素 | 搜索目的 | 搜索词参考（用实际日期替换） |
|------|----------|---------------------------|
| AI 最新重磅动态 | 让客户知道 AI 进展有多快 | "AI 最新发布 {当前年月}" / "大模型 最新动态" |
| AI 发展趋势与预测 | 让客户感到窗口期正在关闭 | "AI 发展趋势 {当前年}" / "AI 智能体 {当前年}" |
| 生产力数据 | 用权威数字证明 AI 的效果 | "AI 生产力提升 报告" / "AI 效率提升 数据" |
| 行业 AI 案例 | 让客户看到同行已经在用 | "{客户行业} AI 应用 {当前年}" |
| 竞品动态 | 制造"别人已经动手了"的焦虑 | "{客户行业} 数字化转型 案例" |

**关键规则：**
- 每次生成都必须重新搜索，不要复用上次的结果
- 搜索结果要用最新的事实和数据，不是模板话
- 数据标注来源（如"麦肯锡 2025 报告"），增强可信度
- 如果搜索不到足够新的内容，用次新的数据并标注时间

**呈现方式：**
- 用数据卡片、时间线、对比图等可视化形式
- 紧迫感来自真实数据，不是空喊口号
- 把客户行业的数据和 AI 全局数据结合，让客户觉得"这事跟我有关"

## 第三层：形象包装

让客户觉得"这是专业团队做的"，第一眼就想继续看。

**设计原则：**
1. **数据说话**：每个结论有数字支撑，不用空话
2. **视觉冲击**：进度条、对比图、指标卡片，客户一眼看到"赋能前 vs 赋能后"
3. **业务语言**：标题用客户听得懂的话，技术术语用简短括号说明
4. **一页抓住重点**：前 3 屏必须包含：客户是谁、有什么问题、我们怎么解决、效果怎样
5. **专业商务风格**：配色协调、排版精致、打印效果好

**多页要求：**
- L1 泛销售：5-8 页
- L2 行业/企业：8-12 页
- L3 企业内部：10-15 页
- L4 渠道合作：5-8 页
- L5 学校合作：视情况

**形象包装不是浮夸。** 核心是让客户 3 秒内看到价值、30 秒内想进一步聊。

## 第四层：市场化描述（CTA）

CTA = Call to Action，行动号召。每份材料的终极目的不是"让人看完"，而是"让人行动"。

**CTA 设计原则：**
- 明确告诉接收者下一步该做什么
- 让接收者觉得行动对他有利（"这事搞成了，对我有好处"）
- 每种场景的 CTA 不同，必须匹配场景

**各场景 CTA：**

| 场景 | 目标受众 | CTA 目标 | 典型引导语 |
|------|----------|----------|------------|
| L1 泛销售 | 任何人 | 想深入了解，转给更具体的人 | "联系我们获取专属方案" / "扫码了解详情" |
| L2 行业/企业 | 行业中间渠道 | 转发给该领域下游联系人 | "转发给贵司 XX 负责人，探讨落地可能" |
| L3 企业内部 | 已沟通的对接人 | 拿去给领导申请预算 | "已具备启动条件，建议安排启动会" |
| L4 渠道合作 | 渠道商 | 直接拿去招生 | "立即开班，名额有限" / "联合招生中" |
| L5 学校合作 | 学校/机构 | 确认合作并启动 | "合作方案已就绪，期待确认" |

**CTA 放在哪里：**
- 材料末尾（必须有）
- Hero 区下方（可选，强化行动感）
- 关键数据页之间（可选，自然引导）

---

## 措辞规范

> 方案面向客户领导，每个用词都要体现专业感和价值感。

| 不要说 | 要说 |
|--------|------|
| AI培训方案 | AI赋能方案 / AI提效方案 / 组织智能升级方案 |
| AI培训 | AI赋能 / AI能力建设 / 智能化升级 |
| 教 / 教程 | 赋能 / 实战 / 能力跃迁 |
| 课程 | 赋能模块 / 实战单元 |
| 学员 | 参训人员 / 赋能对象 |
| 上课 | 赋能实战 / 密集实操 |
| 学AI | 掌握AI生产力 / 建立AI工作方法论 |

核心原则：客户领导看到标题就觉得"这是在帮我的团队升级"，而不是"又要派人去上课"。

**营销话术原则：**
- 用具体数字代替空泛描述（"效率提升 3 倍" 而不是 "提升效率"）
- 紧迫感来自搜索到的真实数据，不是固定口号
- 用搜索结果中的行业案例来制造"同行已经在行动"的焦虑
- 话术要跟随时代，每次生成都用最新的行业语言
