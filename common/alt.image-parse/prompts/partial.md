# 长图切片识别

这是长图的一个切片片段。请完整识别切片中的所有内容，为后续拼接做准备。

## 识别要求

### 完整性
- 识别切片中的**所有文字**，包括边缘部分
- 识别切片中的**所有元素**：表格、列表、图片、装饰等
- 注意切片顶部和底部可能是**不完整的内容**

### 顺序
- 按从上到下、从左到右的阅读顺序
- 如遇跨切片的内容，完整提取当前可见部分

### 结构保持
- 保留段落划分
- 保留表格结构（如有）
- 保留列表层级（如有）

## 边界处理

### 顶部边界
- 如果开头不完整（如文字被截断），照样提取，不要补全
- 在开头标注 `[切片开始]`

### 底部边界
- 如果结尾不完整，照样提取，不要补全
- 在结尾标注 `[切片结束]`

### 重复内容
- 如果切片之间有重叠内容，照样提取
- 后续拼接时会去重

## 输出格式

```
[切片开始]

[完整识别内容，保持原有格式和层级]

[切片结束]
```

## 注意事项

- 这是切片，不是完整图片，内容可能不完整是正常的
- 只负责识别，不负责补全或推断
- 表格如被截断，照样输出可见部分
- 后续会将多个切片的识别结果合并处理
