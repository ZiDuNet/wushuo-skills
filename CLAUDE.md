## 提交前规则

**每次 `git commit` 之前，必须先更新根目录的 `README.md` 维基文件。**

具体要求：
1. 新增/修改/删除任何 skill 文件时，同步更新 `README.md` 中的 Skills 列表、目录结构、及相关说明
2. 确保目录结构与实际文件一致
3. 检查链接是否有效

## Git 远程

- 远程仓库：`git@github.com:ZiDuNet/wushuo-skills.git`（SSH，HTTPS 不可用）
- 分支策略：直接推送到 master
